# v_zh_narr_007

## 音色定位

- 分组：narrator（旁白）
- 档案：male（男声） / senior（老年） / human（人类/常规人物） / zh-CN
- 声音质感：gravel（沙哑低沉）
- 默认语气：authoritative（权威）
- 适配角色：elder_recollection（elder=长者, recollection=回忆）、oral_history（oral=口述, history=历史）
- 不推荐场景：child_dialogue（child=儿童, dialogue=对白）
- 备注：不适合儿童角色。

## 文件约定

- `voice.json`：从音色库导出的锁定元数据。
- `sample.txt`：试听合成文本。
- `sample.voice.txt`：VoxCPM2 音色控制提示。
- `sample.controls.json`：结构化控制参数。
- `sample.wav`：VoxCPM2 生成的无损试听音频。
- `sample.mp3`：由 WAV 转出的便携试听音频。

## VoxCPM2 控制提示

`sample.voice.txt` 保存本音色实际用于 VoxCPM2 合成的原始控制提示。该文件面向程序调用，人工阅读优先看上面的中英对照音色定位。
