# v_zh_ext_096

## 音色定位

- 分组：extension（扩展音色）
- 档案：neutral（中性） / ageless（无年龄感） / nonhuman（非人智慧体） / zh-CN
- 声音质感：ultra_clean（ultra=极致, clean=干净）
- 默认语气：detached（疏离）
- 适配角色：higher_dim_ai（higher=高维, dim=维度, ai=人工智能）、cosmic_narrator（cosmic=宇宙感, narrator=旁白）
- 不推荐场景：earthy_daily_dialogue（earthy=朴实接地气, daily=日常, dialogue=对白）
- 备注：高维中性叙述。

## 文件约定

- `voice.json`：从音色库导出的锁定元数据。
- `sample.txt`：试听合成文本。
- `sample.voice.txt`：VoxCPM2 音色控制提示。
- `sample.controls.json`：结构化控制参数。
- `sample.wav`：VoxCPM2 生成的无损试听音频。
- `sample.mp3`：由 WAV 转出的便携试听音频。

## VoxCPM2 控制提示

`sample.voice.txt` 保存本音色实际用于 VoxCPM2 合成的原始控制提示。该文件面向程序调用，人工阅读优先看上面的中英对照音色定位。
